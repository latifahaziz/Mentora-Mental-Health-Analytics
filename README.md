## Mentora – Predictive System for Workplace Mental Health

Mentora is a data-driven system designed to proactively identify employees who may require mental health support in high-pressure work environments. The project leverages machine learning techniques to provide actionable insights that help organizations foster a healthier workplace.

---

## 🚀 Project Overview

- **Problem:** Traditional workplace mental health approaches are reactive, often leaving employee needs unaddressed.
- **Solution:** A predictive analytics system that uses historical employee data to forecast mental health support requirements.
- **Impact:** Developed a machine learning model achieving approximately **84% accuracy**.

---

## 🛠️ Technical Stack

- **Language:** Python  
- **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib  
- **Tools:** Jupyter Notebook  
- **Models Used:** Logistic Regression, Random Forest, Support Vector Machine (SVM)

## 📊 Key Results

- Achieved strong model performance with balanced precision and recall
- Identified key factors influencing mental health needs, such as:
